## 다이오드
### P형, N형 반도체
- 도체 : 전류를 잘 흐르게 하는 물질.
- 부도체 : 전류를 잘 흐르지 않게 하는 물질.
- 반도체 : 어떤 조건에 맞으면 전류가 흐르고 조건에 맞지 않으면 전류가 흐르지 않음.
- P형 반도체

Ge(게르마늄) + Si(규소/실리콘) + **3가 원소**<br>
전공이 더 많음. 
- N형 반도체

Ge + Si + **5가 원소**<br>
전자가 더 많음<br>
![p형과 n형 반도체의 구조](https://lokigem.github.io/docs/assets/img/CS/디지털공학개론/p형과_n형_반도체의_구조.png)

### 다이오드 동작
- P+ / N- 상태일 때 : 전류가 흐름.
- N+ / P- 상태일 때 : 전류가 흐르지 않음.
